# LLM Crystal Structure Prediction Benchmark

Benchmark large language models on crystal structure prediction (CSP) tasks using the CrystalFormer benchmark datasets.

## Overview

This repository evaluates LLMs' ability to generate accurate crystal structures in CIF format. Generated structures are relaxed using ORB v3 MLFF and compared against ground truth using pymatgen's `StructureMatcher`.

Ground truth structures are from the [CrystalFormer-CSP](https://arxiv.org/abs/2512.18251) benchmark:
- **Dataset I**: 40 common materials (2–104 atoms per unit cell)
- **Dataset II**: 50 complex materials (more challenging)

## Methodology

1. **CIF Generation**: LLM generates CIF from chemical formula + atom count hint
2. **Filtering**: Remove disordered structures and >100 atom cells
3. **Relaxation**: ORB v3 MLFF (FIRE optimizer, fmax=0.1, 500 steps)
4. **E-hull Calculation**: Energy above convex hull using MP2020 compatibility
5. **Structure Matching**: `StructureMatcher(comparator=ElementComparator())`

### Prompt Template

```
System: You are an expert crystallographer. Generate accurate CIF files.

User: Generate a complete CIF (Crystallographic Information File) for the most stable
crystal structure of {formula}. The conventional unit cell contains {n_atoms} atoms.
Output ONLY the CIF content between ```cif and ```.
Include all necessary fields: cell parameters, space group,
atom sites with fractional coordinates.
```

**API Parameters**: Temperature 0.2, max_tokens 4096

## Repository Structure

```
results_ds{1,2}_{model}/     # Raw CIF files generated by LLMs
relaxed_ds{1,2}_{model}/     # Relaxed structures + E-hull data
match_results/               # Structure matching analysis CSVs
run_llm_csp.py              # Main benchmark script
```

## Results Summary

### Dataset I (40 common materials)

| Model | Parsed | Struct Match | SPG Match |
|-------|--------|-------------|-----------|
| Claude-4.6-Opus | 27/40 | 15/40 (37.5%) | 14/40 (35.0%) |
| claude-opus-4-6 (old API) | 28/40 | 12/40 (30.0%) | 12/40 (30.0%) |
| gemini-3-pro-preview-thinking | 25/40 | 12/40 (30.0%) | 12/40 (30.0%) |
| GPT-5.4 | 20/40 | 2/40 (5.0%) | 3/40 (7.5%) |
| gpt-5-chat (old) | 30/40 | 1/40 (2.5%) | 0/40 (0.0%) |

### Dataset II (50 complex materials)

| Model | Parsed | Struct Match | SPG Match |
|-------|--------|-------------|-----------|
| Claude-4.6-Opus | 32/50 | 4/50 (8.0%) | 4/50 (8.0%) |
| GPT-5.4 | 41/50 | 1/50 (2.0%) | 6/50 (12.0%) |

**Notes:**
- "Parsed" = LLM generated valid CIF with correct stoichiometry
- "Struct Match" = Structure matches ground truth via StructureMatcher
- "SPG Match" = Space group number matches ground truth

## Usage

```bash
# Setup
pip install pymatgen mp-api tqdm pandas

# Set API credentials
export API_BASE_URL="https://your-api-endpoint"
export API_KEY="your-api-key"

# Run benchmark on Dataset I
python run_llm_csp.py \
    --dataset path/to/Dataset_I_II_III_with_predictions.pd \
    --dataid 1 \
    --model "model-name" \
    --output results_ds1_mymodel

# Skip existing CIF files (for incremental runs)
python run_llm_csp.py \
    --dataset path/to/Dataset_I_II_III_with_predictions.pd \
    --dataid 2 \
    --model "model-name" \
    --output results_ds2_mymodel \
    --skip-existing
```
